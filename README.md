# üè¶ Bank Account Management System (Java, OOPs)

---

##  Features
- Create Savings or Current Accounts  
- Deposit and Withdraw money  
- Display account details  
- Calculate interest for savings accounts  
- Demonstrates OOP concepts in an easy, real-world example  

---

##  Concepts Used
| Concept | Description |
|----------|--------------|
| **Encapsulation** | Private variables `balance`, `accountHolder` with getters and setters |
| **Inheritance** | `SavingsAccount` and `CurrentAccount` inherit from `BankAccount` |
| **Polymorphism** | `calculateInterest()` method overridden in subclasses |
| **Abstraction** | `BankAccount` declared as an abstract class |
